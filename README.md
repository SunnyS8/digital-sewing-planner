# Digital Sewing Planner / Цифровой швейный планер

Web-приложение для ведения швейных проектов, хранения мерок, генерации персональной фигуры, визуального планирования моделей с наложением ткани и выкройки на силуэт.

## Возможности

- **Мерки** — ввод и хранение индивидуальных замеров (20+ параметров)
- **Фигура (Croquis)** — визуальная SVG-модель тела, построенная по вашим меркам
- **Проекты** — управление швейными проектами со статусами (планирование → раскрой → пошив → примерка → готово)
- **Ткани** — библиотека с загрузкой фото, Drag & Drop
- **Выкройки** — загрузка эскизов и привязка к проекту
- **Примерочная** — наложение выкроек на фигуру с возможностью поворота, масштабирования, регулировки прозрачности и наложения текстуры ткани
- **Экспорт** — PDF, PNG, SVG

## Технологии

React 19 · TypeScript 6 · Vite 8 · Tailwind CSS 4 · Fabric.js 6 · Zustand · react-hook-form · Zod · i18next · jsPDF · html2canvas

## Разработка

```bash
npm install
npm run dev
```

Сборка:

```bash

```
